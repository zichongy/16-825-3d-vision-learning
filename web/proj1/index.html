5.3 Implicit Surfaces

Trade-offs between rendering as a mesh vs a point cloud:

  • Speed: point clouds are faster since they simply draw points instead of rendering actual surfaces.
  • Quality: mesh provides better rendering quality overall.
  • Easy of use: for use-cases such as scanning the real world, point clouds are easier to create. But in some cases we need complete surface representation like finite element analysis, where meshes are easier in terms of input.
